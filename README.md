# news-card-studio-pro
Ultimate video and image news card generator.

## Local development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open the URL shown in the terminal.

## Build for production

```bash
npm run build
```

## Vercel deployment

This repository is configured for Vercel with `vercel.json`. Deploy with:

```bash
vercel deploy --prod
```
